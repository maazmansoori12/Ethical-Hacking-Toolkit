# 🛡️ Ethical Hacking Toolkit

A modern web-based cybersecurity toolkit built with **Python**, **Flask**, **HTML**, **CSS**, and **JavaScript**. This toolkit provides multiple security and networking utilities through a clean, hacker-themed dashboard.

---

## 🚀 Features

- 🔐 Password Generator
- 🔍 Password Strength Checker
- 🌐 DNS Lookup
- 📡 Port Scanner
- 🌍 IP Lookup
- 📧 Email Header Analyzer
- 📄 HTTP Header Analyzer
- 🔒 SSL Certificate Checker
- 🤖 Technology Detector
- 📋 Hash Generator (MD5, SHA1, SHA256)
- 🔄 Base64 Encoder / Decoder
- 📱 QR Code Generator
- 🤖 Robots.txt Checker
- 🗺️ Sitemap Finder
- 📊 Network Scanner
- 🌐 Subnet Calculator
- 📜 Logs Viewer
- And more...

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/maazmansoori12/Ethical-Hacking-Toolkit.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
